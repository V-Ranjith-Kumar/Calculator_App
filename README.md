# Calculator_App

# 🧮 Flutter Calculator App

A simple and responsive calculator built with **Flutter**.  
This project demonstrates Flutter’s cross‑platform capabilities and can run on Android, iOS, Web, Windows, macOS, and Linux.

---

## 🚀 Getting Started

### 1. Prerequisites
- Install [Flutter SDK](https://docs.flutter.dev/get-started/install) (latest stable version).
- Install [VS Code](https://code.visualstudio.com/) or Android Studio.
- Ensure you have Git installed.
- Set up a device/emulator (Android, iOS, or use Chrome/Web).

Check your Flutter installation:

bash

flutter doctor

2. Clone the Repository
bash

git clone https://github.com/V-Ranjith-Kumar/Calculator_App.git
cd Calculator_App
4. Install Dependencies
bash

flutter pub get

5. Run the App
Choose your platform:

Android/iOS:

bash

flutter run
Web:

bash
flutter run -d chrome
Windows/macOS/Linux:

bash
flutter run -d windows
📱 Features
Basic arithmetic operations (+, −, ×, ÷).


Clean and responsive UI.

Cross‑platform support (mobile, web, desktop).

🛠 Project Structure
lib/ → Main Dart source code.

android/, ios/, web/, windows/, macos/, linux/ → Platform‑specific code.

pubspec.yaml → Dependencies and assets.
